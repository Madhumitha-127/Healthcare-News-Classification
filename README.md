# Healthcare-News-Classification
## Project Overview

This project classifies healthcare news articles into four categories:

* Disease Outbreak
* Health Policy
* Medical Research
* Pharma Drug

The system uses Natural Language Processing (NLP) techniques and Machine Learning for automatic classification.

## Technologies Used

* Python
* Flask
* Scikit-learn
* TF-IDF Vectorization
* Logistic Regression
* HTML
* CSS

## Features

* News classification
* Confidence score prediction
* Interactive web interface
* NLP preprocessing

## Dataset

Custom healthcare news dataset containing 5000 records.

## Run

```bash
pip install -r requirements.txt
python train_model.py
python app.py
```
